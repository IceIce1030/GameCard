# 🌊 GameCard - 海底遊戲樂園

歡迎來到 **GameCard**！這是一個基於 ASP.NET Core 打造、以「深海風情」為視覺主題的網頁遊戲平台。本專案將經典的卡牌、益智與對戰類遊戲，與豐富的海洋視覺元素（如珊瑚藍漸層、貝殼、海浪等）完美結合，帶給玩家沉浸式的輕鬆娛樂體驗。

---

## 🎮 核心遊戲與功能

本平台目前收錄了多款各具特色的網頁遊戲與功能頁面，依類型分為「休閒遊戲」、「卡牌系統」與「平台資訊」三大區塊：

### 🎯 休閒遊戲

1. **🐚 撲克牌比大小 (`/CardGame`)**
   - 經典的撲克牌機率遊戲，考驗你的直覺與運氣。
2. **🍼 養育小寶寶 (`/Baby`)**
   - 療癒系的小寶寶養成模擬，體驗日常育兒的酸甜苦辣。
3. **🎲 育兒大富翁 (`/BabyTycoon`)**
   - 將經典大富翁與育兒情境結合，透過擲骰子應對各種育兒隨機事件。
4. **🌐 大富翁連線版 (`/BabyTycoonOnline`)**
   - 支援線上多人連線，與好友一起在網路上進行激烈的育兒大富翁對決。
5. **📈 瘋狂股市對決 (`/StockBattle`)**
   - 模擬股市波動的刺激對決，比拚誰的投資眼光更精準。

### ⚔️ 卡牌系統

6. **⚔️ 抽卡召喚 (`/Cards`)**
   - 透過抽卡機制召喚各式卡牌，蒐集你的專屬陣容。
7. **🎒 卡牌背包 (`/Backpack`)**
   - 管理與檢視你所擁有的全部卡牌收藏。
8. **🥊 競技對戰 (`/Battle`)**
   - 運用手上的卡牌進行單機競技對戰。
9. **🌐 線上對戰 (`/BattleOnline`)**
   - 支援線上連線，與其他玩家進行即時卡牌對戰。

### 📖 平台資訊

10. **📖 關於我 (`/About`)**
    - 介紹平台理念、開發背景與團隊資訊。
11. **🛡️ 隱私權 (`/Privacy`)**
    - 完善的平台隱私權益說明頁面。

## 🎨 視覺與技術特色

- **深海漸層導覽列**：採用 `#053047` 到 `#0a6e8c` 的極致深海漸層配色，搭配發光亮點，營造神祕的氛圍。
- **動態懸浮商標**：Logo 附帶微動態的波浪懸浮效果（`brandFloat` 動畫），視覺更加靈動。
- **主題式 Emoji 圖標**：精選 🌊、🐚、🍼、🎲、🌐、📈、⚔️、🎒、🥊、📖、🛡️ 等視覺錨點，讓選單直覺易讀。
- **分區式導覽選單**：選單以分隔線（`ocean-sep`）將遊戲、卡牌系統與平台資訊清楚劃分，瀏覽更有條理。
- **滿版無縫版面**：主內容區（`.ocean-main`）打破傳統框架限制，讓海底背景鋪滿全螢幕。

## 🛠️ 開發技術棧 (Tech Stack)

- **後端架構**：ASP.NET Core (Razor Pages)
- **前端框架**：Bootstrap 5 / jQuery
- **樣式客製**：CSS3 Flexbox / CSS3 Animation / CSS Gradient
- **連線技術**：SignalR (用於大富翁連線版與線上對戰)
